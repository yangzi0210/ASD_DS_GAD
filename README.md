# ASD_DS_GAD
 基于图异常检测等机器学习技术以及 ABIDE 数据集进行 ASD 患者的诊断

## Requirements
```

torch == 1.8.1+cu102

torch-cluster == 1.5.9

torch-geometric == 1.7.0

torch-scatter == 2.0.7

torch-sparse == 0.6.10

sklearn

nilearn

```
